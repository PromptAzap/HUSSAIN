# 🗂️ البيانات الوصفية لملفات المشروع (Files Metadata Dictionary)

> [!NOTE]
> هذا القاموس الوصفي يعمل كدليل منهجي لمهندسي البيانات والمطورين لفهم الغرض من وراء كل وحدة برمجية أو مخزن بيانات داخل مشروع المحرك الهجين للأنطولوجيا الإسلامية.

---

### جدول البيانات الوصفي الكامل

| اسم الملف | المسار | النوع | الوصف (الهدف التقني والمعرفي) |
| :--- | :--- | :--- | :--- |
| `hybrid_search.py` | `./` | سكربت مركزي | النواة الأساسية للمحرك الهجين للبحث الدلالي والروتينج التلقائي (Semantic & Hard-link Routing) مع دوال إنشاء الفهرسة. |
| `quran_roots_dual_v2.sqlite` | `./` | قاعدة بيانات | مخزن علائقي يحتوي على نصوص القرآن الخالصة والعثمانية، إضافة للجذور الصرفية واللغوية للكلمات للربط المعرفي. |
| `unified_ontology.ttl` | `./` | Knowledge Graph | الإطار العام للمشروع بصيغة Turtle، يحتوي على شبكة المفاهيم وعلاقاتها الهرمية ليواكب معايير الويب الدلالي المنطقية (Semantic Web). |
| `concept_ayah_mapping.json` | `./` | ملف بيانات (قاموس) | ملف "الروابط الثابتة" (Hard-links). يقوم بربط مقصدي ثابت بين المُعرف الفلسفي للمفهوم في الأنطولوجيا وبين أقوى الآيات القرآنية الدالة عليه. |
| `requirements.txt` | `./` | ملف نصي | قائمة بمكتبات البيئة اللازمة لتشغيل المحرك الدلالي (مثل `sentence-transformers`, `chromadb`). |
| `generate_ontology.ps1` | `./` | سكربت PowerShell | أداة مخصصة كُتبت لأتمتة تجميع وتوليد ملفات الأنطولوجيا أو تجديد تصنيفها السريع في بيئة ويندوز. |
| `README.md` | `./` | مستند تعريفي | الواجهة التعريفية الخفيفة التي تقدم لمحة مباشرة وسريعة عن المحرك وكيفية التثبيت. |
| `project_documentation.md` | `./` | مستند دلالي | الشرح الفني التفصيلي القصير للمحرك الذي يوضح كيف يعمل البحث الهجين ودورة التوجيه الذاتي للطلب. |
| `HUSSAIN_Master_Documentation.md`| `./` | مستند المعمارية | الوثيقة المرجعية الشاملة (Master Document) التي تؤطر الفلسفة والتقنية والممارسات الآمنة والاحتياجات الضمنية للتوسع نحو API. |
| **مجلدات فرعية:** |
| `archive_v1/` | `./archive_v1` | مجلد بيانات | يضم هيكليات JSON القديمة السابقة لكل سلسلة مفهوم (مثل المائدة وغيرها) وسكربتات استخراج البيانات (`extract_names`, `build_bridge`). يُشكل الخام الذي تتغذى منه الفهرسة الدلالية حالياً. |
| `ماركداون/` | `./ماركداون` | مجلد نصوص خام | نصوص وتفريغات خام لعدة ملفات ماركداون تتضمن الدروس والمفردات من عدة سلاسل معرفية، وهي مسودات العمل المنقولة يدوياً. |
| `python-embed/` | `./python-embed` | بيئة تشغيل مستقلة | بيئة بايثون المصغرة ذاتية الاحتواء (Embedded Python) لضمان التشغيل المستقل للمحرك بدون الحاجة بالضرورة لتثبيت بايثون على جهاز العميل. |
